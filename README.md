# Sign Language Detector

This project detects American Sign Language (ASL) alphabets using a webcam.

## Features
- Real-time hand detection using MediaPipe
- Landmark-based ASL alphabet recognition
- Machine Learning model (RandomForest)
- Live webcam prediction

## Requirements
- Python 3.9+
- OpenCV
- MediaPipe
- NumPy
- scikit-learn
- joblib
